Projects

All Signal Processing Deep Learning Others

Chest Xray Analysis Using Deep Learning

Trained models using dataset of 100,000 Chest X-Ray images to identify 14 different diseases (labels) in a given Chest X-Ray image.

Customer Profiling and Language Detection Using Real-Time Call Center Data

Given are recording between customer and agent, and aim is to predict features of the customer.We do this by extracting MFCC features, VAD and then training a SVR and DNN using these features. We feed First Order Statistics calculated from a GMM-UBM to the SVR/DNN Model.

Despeckiling of OCT Images using Auto-encoders

OCT images contain certain multiplicative noise. Using deep learning algorithms, in particular auto-encoders, we aim to remove this noise and reconstruct the image keeping the information intact.

Multi-SIM Multi Standy Resource Allocation

The project focuses on MAC layer with respect to nding allocation pattern (created by the different scheduling algorithms) for single SIM and verifying the same for multi SIM model with pause and resume scenario.

Internship at World Wide Fund

Lead the entry of data regarding different crops, the project was based in Chhindwara district of Madhya Pradesh under the Sustainable Agriculture Division of WWF.

Doppler Processing Algorithm for Clutter Supression and MTI

MTI or Moving target indication is a problem in radar systems which we solved using Doppler shift so that it can be distinguished from clutter (stationary objects).

3D modelling using blender and integration with Spark AR

The project involves modelling in Blender, and using Spark AR for adding in effects for augmented reality. The end goal is to make a filter.
